# 📊 RetailIQ — Retail Sales & Profit Dashboard

A fully interactive **retail analytics dashboard** built using **vanilla HTML, CSS, and JavaScript** with the Superstore dataset.  
It visualizes key business KPIs like sales, profit, customer segments, regional performance, and product-level insights.

## ✨ Features

### 📈 Analytics Dashboard
- Sales & Profit trend analysis (monthly)
- Profit breakdown by category
- Regional sales distribution
- Customer segment insights

### 📊 Interactive Visualizations
- Line chart (Sales & Profit trends)
- Doughnut chart (Region-wise sales)
- Bar chart (Category performance)
- Pie chart (Customer segmentation)
- Horizontal bar chart (Sub-category performance)

### 🔍 Filters
- Year-wise filtering (2014–2017)
- Category filtering
- Region filtering
- Dynamic insight updates

### 📋 Data Table
- Top products ranked by sales
- Profit margin visualization
- Category-wise breakdown

---

## 🧰 Tech Stack

- HTML5 — structure
- CSS3 — styling & layout system
- JavaScript (ES6) — logic & interactivity
- Chart.js — data visualization
- Google Fonts — typography
